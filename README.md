Este repositório contém uma **calculadora de notas escolares**, onde é possível inserir suas notas e descobrir automaticamente:

- sua **média final**
- se você foi **aprovado ou reprovado**
- **quanto falta para passar**, caso não tenha atingido a média mínima

# ↪︎ Sobre o projeto

O objetivo deste projeto é facilitar o cálculo das notas durante o **ano letivo**, permitindo que o estudante veja rapidamente sua situação escolar.

O sistema considera os **pesos das notas**, calcula a média automaticamente e mostra o resultado de forma clara.

---

# ↪︎ Funcionalidades

Cálculo automático da média  
Indicação de **aprovado ou reprovado**  
Mostra **quanto falta para atingir a média mínima**  
Botão para limpar as notas  
Interface moderna e simples
